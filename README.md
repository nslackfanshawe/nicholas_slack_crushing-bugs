# nicholas_slack_crushing-bugs
This is the submission for the crushing bugs assignment for multi-media authoring 2.

## Description
- This is a drag and drop game that allows the user to drag labels with various body parts on them and drop them into the correct place on the body.

## Features
- Drag and drop functionality
- The drop zones only accept one label.
- Changing colours when hovering
- A reset button that resets the whole game.

## Technologies Used
- HTML
- CSS
- JS

## Setup Instructions
- Download or clone respository
- Open the project folder
- Open index.html 
- No aditional setup is required
